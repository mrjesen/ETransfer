### 一个简单的P2P聊天室 支持文件传输 支持屏幕共享

## [DEMO](https://et.aman.icu)

## 服务端（server目录）

### run
```shell
node  ./server/server.js
```

## 客户端 （client目录）
vue3 + primevue

### install
```shell
yarn install
```

### run
```shell
yarn serve
```

### build
```shell
yarn build
```

### Lints and fixes files
```shell
yarn lint
```
